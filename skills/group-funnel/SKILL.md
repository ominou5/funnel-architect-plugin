# Group Funnel

A high-converting 2-page funnel optimized for driving signups into a community group. Works with any platform: **Skool, Whop, Facebook Groups, Discord, Circle, Mighty Networks**, or any community URL.

> **Trigger keywords:** If the user mentions "Skool", "Whop", "Facebook group", "community group", "Discord server", "Circle community", "membership group", "free group", or "paid group" — this is the funnel to use.

## Funnel Flow

```
Traffic → Opt-In Page → Thank You Page
              │                │
         [Email Form]    [Deliver Lead Magnet]
                              [Community CTA → Group URL]
                              [Social Share Buttons]
```

## Pages

### Page 1: Opt-In Page

**Purpose:** Capture the visitor's email in exchange for a lead magnet, then redirect to the community-focused thank-you page.

**Required Sections (in order):**
1. **Top bar** — Announcement strip highlighting the free offer
2. **Hero** — Eyebrow badge, benefit-driven headline, sub-headline targeting the audience, fade-in animations
3. **Video mockup** — Stylized card with play button, content title, brand avatar + badge
4. **Bullet benefits** — 4–5 bullet points with check-mark icons
5. **Opt-in form** — Name + email fields with icon indicators, prominent CTA button with pulse animation
6. **Social proof bar** — Stacked avatars, star rating, member count
7. **Privacy line** — Zero-spam guarantee inside the form card

**CTA Examples:**
- "Yes! Send Me the Free Mini-Course →"
- "Get Instant Access to the [Lead Magnet] →"
- "Join [Number]+ [Audience] — Send It Now"

**What to REMOVE:**
- Navigation menu (no escape routes)
- Multiple competing CTAs
- Pricing or payment content
- External links above the fold

### Page 2: Thank You Page

**Purpose:** Confirm opt-in, deliver the lead magnet, and drive the visitor into the community group.

**Required Sections (in order):**
1. **Success bar** — Green confirmation strip
2. **Confirmation hero** — Animated check circle, headline, email check instructions
3. **Video placeholder** — Stylized video card with play button, loading dots, content title
4. **Community CTA card** — Eyebrow badge, headline selling the community, description, 4-perk grid, CTA button linking to community group, free-to-join note
5. **Social share section** — Twitter/X share button + copy-link button

**Community Perks Grid (customize all 4):**
- Perk 1 — e.g., "Live Community" + description
- Perk 2 — e.g., "Resource Library" + description
- Perk 3 — e.g., "Weekly Challenges" + description
- Perk 4 — e.g., "Full Course Library" + description

## Conversion Benchmarks

| Metric | Good | Great | Elite |
|---|---|---|---|
| Opt-in rate | 25% | 35% | 50%+ |
| Email open rate (welcome) | 40% | 55% | 70%+ |
| Thank-you → group join | 15% | 25% | 40%+ |
| Thank-you social share | 3% | 8% | 15%+ |

## A/B Test Recommendations

1. **Headline** — Test problem-focused vs. result-focused
2. **CTA button text** — Test "Get Instant Access" vs. "Yes! Send It Now"
3. **Video mockup** — Test mockup vs. embedded real video
4. **Bullet count** — Test 3 bullets vs. 5 bullets
5. **Social proof** — Test avatar stack vs. testimonial quote
6. **Community CTA placement** — Test above vs. below video on thank-you page
