# Tripwire Funnel

Turn leads into buyers with a low-friction, low-risk first purchase. Then upsell.

## Flow

```
Ad / Content → Opt-In → Tripwire Offer ($7–$47) → Order Bump → Upsell → Thank You
```

| Page | Purpose | Price |
|---|---|---|
| Opt-In | Capture email with lead magnet | Free |
| Tripwire | Irresistible low-ticket offer | $7–$47 |
| Order Bump | Add-on shown on checkout | +$17–$37 |
| Upsell | Core product or bundle | $97–$497 |
| Downsell (optional) | Lighter version if upsell declined | $47–$197 |
| Thank You | Deliver everything | — |

## Tripwire Offer Criteria
1. **Solves one specific problem** fast
2. **High perceived value** relative to price (10x value)
3. **Digital delivery** — no shipping friction
4. **Complements the core product** — taste of what's to come
5. **Easy to say yes** — removes buyer hesitation

### Tripwire Ideas by Niche
| Niche | Tripwire Offer |
|---|---|
| Marketing | 50 Email Subject Line Templates — $17 |
| Fitness | 7-Day Meal Plan + Grocery List — $9 |
| SaaS | Lifetime access to one micro-tool — $27 |
| Coaching | 1-Hour Recorded Masterclass — $19 |
| E-Commerce | Sample pack / trial kit — $12 |

## Order Bump Copy Template
```
✅ ADD THIS: {{BUMP_PRODUCT}} (just ${{BUMP_PRICE}} more)
Most customers add this — [one-sentence benefit].
```

## Upsell Page Structure
1. **Congratulations** message (positive momentum)
2. **"Wait — your order is not complete"** pattern interrupt
3. **Introduce the upsell** with value-first framing
4. **One-click purchase** button (no re-entering payment)
5. **"No thanks" link** below (never hide the decline option)

## Conversion Benchmarks

| Metric | Target |
|---|---|
| Opt-in rate | > 35% |
| Opt-in → tripwire purchase | > 8% |
| Order bump take rate | > 30% |
| Upsell take rate | > 15% |
| Avg revenue per lead | > $3.50 |

## Economics Example
```
1,000 leads
→ 80 buy tripwire at $17      = $1,360
→ 24 add order bump at $27    = $648
→ 12 buy upsell at $197       = $2,364
──────────────────────────────────────
Total revenue                  = $4,372
Revenue per lead               = $4.37
Ad cost per lead (target)      = $2–3
Result: PROFITABLE from Day 1
```
