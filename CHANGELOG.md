# Changelog

All notable changes to the Sales Funnel Plugin will be documented in this file.

## [1.0.0] - 2025-02-24

### Added
- Plugin manifest and core architecture
- 6 core skills: funnel-strategy, funnel-select, conversion-ux, page-speed, mobile-responsive, design-system
- 15 funnel type skills: optin, webinar, saas, direct-response, ecommerce, tripwire, challenge, application, book, quiz, summit, membership, waitlist, affiliate, multi-step
- 3 cross-cutting skills: funnel-copy, ab-testing, conversion-audit
- 4 deployment skills: deploy-netlify, deploy-vercel, deploy-cloudflare, dns-management
- 3 integration skills: analytics-setup, marketing-stack, cms-integration
- 5 sub-agents: funnel-architect, conversion-optimizer, page-speed-optimizer, copy-doctor, deploy-assistant
- Hooks: async Lighthouse audit, sync mobile check, agent-based final validation, welcome prompt
- Pre-configured MCP servers for GA4, Stripe, HubSpot, Mailchimp, Vercel, Firebase, Supabase
- Utility scripts: mobile-check.sh, lighthouse-audit.js, validate-funnel-structure.js
